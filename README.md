# HR Employee Attrition Analyzer

A data-driven dashboard for analyzing and predicting employee attrition risk using machine learning. Built with Streamlit, it provides interactive visualizations and a Random Forest-based predictor to help HR teams identify at-risk employees and understand key attrition factors.

## Features

- **Attrition Analytics Dashboard** – Visual breakdowns of attrition trends across departments, roles, and demographics.
- **Risk Predictor** – Predict individual employee attrition risk (Low / Medium / High) using a trained Random Forest model.
- **Profile Analysis** – Explore employee profiles and surface patterns linked to turnover.

## Getting Started

### Prerequisites

- Python 3.8+
- pip

### Installation

```bash
pip install -r requirements.txt
```

### Run the App

```bash
streamlit run hr_dashboard.py
```

The dashboard will open in your browser at `http://localhost:8501`.
